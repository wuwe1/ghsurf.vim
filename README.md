ghsurf.vim
==========

Surfing on GitHub with Vim.

Installation
------------

Use your favorite plugin manager.

Using [vim-plug](https://github.com/junegunn/vim-plug):

```vim
Plug 'wuwe1/ghsurf.vim'
```

Command
-------

```bash
# edit file hosted on github
:GHE https://github.com/wuwe1/gh.vim/README.md
```
